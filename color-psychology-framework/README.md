# Color Psychology Framework

A practical knowledge base for understanding color psychology and making informed color decisions for branding, design, and product development.

## Quick Reference

### Color Emotion Matrix

| Color | Primary Emotions | Best For | Avoid When |
|-------|-----------------|----------|------------|
| **Red** | Passion, energy, urgency | Sales, food, entertainment | Healthcare, finance (overuse) |
| **Blue** | Trust, calm, professionalism | Tech, finance, healthcare | Food, urgency-driven sales |
| **Yellow** | Optimism, creativity, attention | Children's products, creative industries | Luxury brands, serious contexts |
| **Green** | Growth, health, nature | Eco-friendly, wellness, finance | Urgency, excitement-driven |
| **Orange** | Enthusiasm, warmth, playfulness | Youth brands, CTAs, food | Luxury, corporate, healthcare |
| **Purple** | Luxury, creativity, wisdom | Premium brands, beauty, spirituality | Budget brands, masculine products |
| **Black** | Sophistication, power, elegance | Luxury, fashion, tech | Children's products, healthcare |
| **White** | Purity, simplicity, cleanliness | Minimalist brands, healthcare, tech | Warmth-seeking contexts |

### The 60-30-10 Rule

```
Primary Color (60%)   → Brand foundation, backgrounds
Secondary Color (30%) → Supporting elements, accents
Accent Color (10%)    → CTAs, highlights, emphasis
```

### Color Temperature Guide

| Warm Colors | Cool Colors | Neutral Colors |
|-------------|-------------|----------------|
| Red, Orange, Yellow | Blue, Green, Purple | Black, White, Gray, Brown |
| Energizing, stimulating | Calming, professional | Balanced, versatile |
| Advance visually | Recede visually | Ground and anchor |

## How to Use This Framework

### For LLM Coaching Sessions
1. Start with `04-llm-integration/decision-trees.md` for guided color selection
2. Reference `02-color-profiles/` for detailed color meanings
3. Use `03-application/industry-color-guide.md` for sector-specific advice

### For Brand Color Selection
1. Define brand personality and values
2. Identify target audience demographics
3. Research industry conventions
4. Select primary color aligned with core emotion
5. Choose complementary secondary and accent colors

### For Design Decisions
1. Consider context and medium
2. Evaluate cultural implications
3. Test color combinations for accessibility
4. Validate emotional response with target users

## Key Principles

1. **Context matters** — The same color evokes different responses in different contexts
2. **Culture influences perception** — Color meanings vary across cultures
3. **Contrast creates hierarchy** — Use color contrast to guide attention
4. **Consistency builds recognition** — Maintain color consistency across touchpoints
5. **Less is more** — Limit palette to 3-5 colors maximum
